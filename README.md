# :books: Install Blynk Server on Raspberry Pi :rocket:

## Quick Install

1) Clone and run this repo

```shell
git clone https://github.com/librarysteve/installblynkpi.git
```
```shell
cd ./installblynkpi
chmod +x ./install.sh
./install.sh
```
